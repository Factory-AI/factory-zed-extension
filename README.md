# Factory Droid - Zed Extension

This extension integrates [Factory's Droid](https://factory.ai) AI coding agent with [Zed](https://zed.dev).

## Installation

1. Open Zed
2. Open the Extensions panel (`Cmd+Shift+X` on macOS, `Ctrl+Shift+X` on Linux/Windows)
3. Search for "Factory Droid"
4. Click Install

## Authentication

After installing, you'll need to authenticate with Factory:

1. Open Zed's Agent Panel
2. Select "Factory Droid" as your agent
3. Follow the authentication prompts

Alternatively, authenticate by starting the CLI via:

```bash
droid
```

Followed by running the `/login` command.

## Requirements

- Zed editor (latest version recommended)
- Factory account ([sign up](https://app.factory.ai))

## Updating

The extension will automatically download the latest Droid binary when you update the extension.

To manually update:

1. Open Extensions panel
2. Find "Factory Droid"
3. Click Update (if available)
